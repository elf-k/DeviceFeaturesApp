## Description
This is a React Native lab project built with Expo.  
The app demonstrates safe and professional usage of native device features such as camera, gallery, GPS, notifications, and haptic feedback.

## Features
- Take a photo using the camera
- Pick an image from the gallery
- Display selected image
- Get current GPS location
- Show local notifications
- Trigger haptic feedback
- Permission handling for all native features

## Setup Instructions
1. Install Node.js and npm
2. Install Expo Go on your mobile device
3. Clone the repository
4. Run the following commands:

```bash
npm install
npx expo start
